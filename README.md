# ⛪ 설교 쇼츠 메이커

유튜브 설교 영상을 자동으로 쇼츠(9:16)로 나눠주는 웹 애플리케이션입니다.

## 기능

- 🎥 유튜브 설교 영상 자동 다운로드
- ✂️ 원하는 길이(분)로 자동 분할
- 📱 쇼츠 형식(9:16)으로 자동 변환
- 📥 분할된 영상 다운로드

## 설치

```bash
pip install -r requirements.txt
```

## 실행

```bash
python app.py
```

로컬: http://127.0.0.1:5000

## 배포 (Vercel)

이 프로젝트는 Vercel에서 무료로 호스팅할 수 있습니다.

1. GitHub에 푸시
2. Vercel 연결
3. 자동 배포 완료

## 필수 라이브러리

- Flask: 웹 서버
- yt-dlp: 유튜브 다운로드
- ffmpeg: 영상 처리
